Your awesome wiki generator

Get Yawg Get Started

Yawg is a static site generator. Yawg generates a directory tree containing the HTML pages for your site from a directory tree of text documents in one of the supported markup formats.

Why Yawg?

Some good reasons to use Yawg.

Easy installation

Download and unpack the binary release tarball. You are ready to run.

Source content formats

AsciiDoc and HTML are the currently supported source markup formats.

Templates

Page templates are used for generating the final HTML pages. Freemarker templates are supported.

 

Diagrams, formulas, more

PlantUML diagrams, LaTeX MathJax formulas, syntax highlithing, tables of contents. All supported out of the box. See our examples.
